## üìã Overview

**Sweet-O-Meter** helps users track daily sugar consumption through automated nutrition label scanning, eliminating tedious manual entry while maintaining data accuracy through intelligent validation.

### The Problem

Traditional manual sugar tracking is time-consuming and creates friction in health monitoring, discouraging consistent dietary awareness. Users need a faster way to track sugar intake without sacrificing accuracy.

### The Solution

An iOS app leveraging Vision Framework OCR to automatically extract sugar values from nutrition labels, combined with manual entry flexibility for validation and edge cases.
